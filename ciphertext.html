Why this page shines

A compact workspace for quick text ciphers, with an intelligent mode swap and live output preview that keeps the cryptography flow smooth.

Caesar Base64 ROT13 Binary Letter number ASCII Vigenère Rail Fence
  • Use Caesar for simple shifts. The key updates automatically.
  • Base64 is great for compact encoding and quick data transport.
  • ROT13 is perfect for reversible obfuscation without a key.
  • Binary and ASCII convert text into numeric forms and back.
  • Letter number maps A→1 and reverses numbered sequences to letters.
  • Vigenère, Rail Fence, XOR, and AES all use keys for more advanced transformations.

Type a phrase, choose a method, then press Transform. Use Swap mode to flip between encoding and decoding for the selected cipher.
