Number Calculator Tips

  • Enter expressions using numbers in the selected From base.
  • Operators: + - * / and parentheses.
  • Use unary minus like: -1010.
  • Digits: 0–9 then A–Z for values 10–35.
  • Base64 is supported when selected as the From base.
  • Division is integer (BigInt) division.
