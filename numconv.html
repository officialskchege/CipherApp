Number Conversion Tips

  • Binary uses only 0 and 1.
  • Octal digits range from 0 to 7.
  • Hexadecimal uses 0–9 and A–F.
  • Digits above 9 are represented by letters A–Z.
  • Base 64 uses A–Z, a–z, 0–9, +, / and padding =.
  • Use "Swap bases" to quickly reverse conversions.
